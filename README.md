## 📌Interpretor 방식
- 소스코드를 기계어로 변역하는 과정 없이 코드를 실행 시점에(코드 한 줄씩) 해석해 컴퓨터가 처리할 수 있도록 함. 간단히 작성 가능하고 메모리를 적게 필요로 하지만 실행속도가 느리다.
