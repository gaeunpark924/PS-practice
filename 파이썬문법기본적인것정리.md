### True/False
아무것도 없으면 False 하나라도 있으면 True
- " " -> True
- "" -> False
- ["",""] -> True
- [] -> False
- [""] -> True
- None -> False

### zip
- 하나로 묶어주는 함수
```Python
a = [1,2,3,4]
b = [100,200,300,400]
print(list(zip(a,b)))
>[(1,100),(2,200),(3,300),(4,400)]
```

### map
- map(함수, 순회 가능한 값)
```Python
def f(n):
   return n**2
print(list(map(f,[2,3,4])))
> [4,9,16]
```
